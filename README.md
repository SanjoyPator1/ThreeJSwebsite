# ThreeJSwebsite

## Installation

Use npm to install the necessary packages.

```bash
npm install
```

## Run locally

Use npm run to run locally.

```bash
npm run dev
```

## See the preview server

To see how it looks in the server.

```bash
npm run serve
```

## BUILD

To build the dist folder

```bash
npm build
```
